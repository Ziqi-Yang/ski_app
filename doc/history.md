# 历史数据

## 接口说明

**需要一个测试接口**  

`post`请求, 接受 `application/json`格式的`header`,
header范例:
```json
{
  "from": "2022/04/01",
  "to": "2022/04/15"
}
```

## 返回数据

示例:
```json
{
  "2022/04/01": [
    {
      "from": "09/48",
      "to": "10/22",
      "score": 80,
      "isFav": true
    },
    {
      "from": "14/42",
      "to": "15/11",
      "score": 83,
      "isFav": true
    }
  ],
  "2022/04/02": null,
  ...,
  "2022/04/15": [
    ...
  ]
}
```

## 字段解释

除了示例中标注的`null`，另外都是`NonNull`的. 另外,`score`是`int`类型
